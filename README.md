# Encryption
Приложение, позволяющее использовать некоторые алгоритмы шифрования

>Поддерживаемые алгоритмы:
  * Шифр Цезаря
  * Шифр Виженера
  * Rot13
  * Гаммирование
  * Квадрат Полибия
  * Тарабарская грамота
